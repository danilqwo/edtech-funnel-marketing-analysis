##  Основные результаты
* **Качество данных (DQ):** Проведена полная проверка на дубликаты, пропуски и логические ошибки. Данные готовы к анализу.
* **Обнаружена проблема:** Выявлено, что **22.7% оплат завершаются ошибкой**.
* **Инсайт:** Пользователи, столкнувшиеся с ошибкой оплаты, редко совершают повторные попытки, что ведет к прямой потере прибыли.
